Sabzineh Enterprise Dashboard — Modern Business Management Dashboard

  A modern enterprise dashboard built entirely with Vanilla JavaScript, HTML5, and Tailwind CSS — without using frontend frameworks.

Live Demo
  https://sabzineh-enterprise-dashboard.vercel.app/

Overview

  Sabzineh Enterprise Dashboard is a modern business management dashboard designed to simulate a real-world enterprise administration panel.
  
  The project demonstrates that complex dashboard interfaces with analytics, inventory management, CRM, reporting, calendars, authentication flows,
  and interactive visualizations can be built entirely using Vanilla JavaScript without relying on frameworks such as React or Vue.
  
  The focus of this project is modular architecture, reusable UI components, scalable state management, premium UI design, responsive layouts, and smooth user experience.
  
  The dashboard follows a custom design system built specifically for the Sabzineh brand using warm earthy colors rather than conventional blue enterprise themes.
